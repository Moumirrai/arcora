# Contributing

## Requirements

- Bun [https://bun.sh/]
- Zig 0.17.0-dev.44+0177cb57c or newer [https://ziglang.org/]

## Development Flow

1. Create a branch from `dev`.
2. Name the branch after the work you are doing.
3. Keep changes small and focused.
4. Before committing, run:

```sh
bun run check
```

This formats the code, runs lint, and runs all tests.
